## How to Use

### In Cursor
1. Open command palette (Cmd/Ctrl + Shift + P)
2. Type the command name (e.g., `/create-plan`)
3. Follow the prompt template

### In Claude Code
1. Reference the command file
2. Copy the prompt template
3. Fill in the placeholders
4. Execute

## Workflow Example

Feature Development Flow:
1. /explore → Understand the codebase area
2. /create-plan → Design the solution
3. /execute → Implement the plan
4. /review → Self-check the code
5. /peer-review → Get feedback
6. /document → Update docs
7. /learning-opportunity → Extract lessons learned
